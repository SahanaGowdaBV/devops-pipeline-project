# 🚀 DevOps Pipeline Project

## 📌 Overview
This project demonstrates an end-to-end CI/CD pipeline using Docker, Kubernetes, and GitHub Actions.

## 🏗️ Architecture
Code → GitHub → CI/CD → Docker → Kubernetes

## ⚙️ Tech Stack
- Node.js
- Docker
- Kubernetes (Minikube)
- GitHub Actions

## 🔄 CI/CD Flow
1. Code pushed to GitHub
2. GitHub Actions builds Docker image
3. Image pushed to Docker Hub
4. Kubernetes deployment updated
